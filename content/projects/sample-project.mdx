## 概要

これはサンプルプロジェクトです。実際のプロジェクトでは、ここにプロジェクトの詳細な説明を書きます。

このファイルは MDX 形式で書かれており、Markdown と React コンポーネントを組み合わせて使用できます。

## 主な機能

- **機能 1**: ユーザー認証とセッション管理
- **機能 2**: データの CRUD 操作
- **機能 3**: リアルタイム通知

## 技術スタック

このプロジェクトでは以下の技術を使用しています：

1. **フロントエンド**: React + Next.js
2. **スタイリング**: Tailwind CSS
3. **型安全性**: TypeScript
4. **デプロイ**: Vercel

## コードサンプル

// TypeScriptのコード例

このプロジェクトを通じて、以下のことを学びました：

- Next.js App Router の使い方

## 次のステップ

このサンプルを削除して、あなた自身のプロジェクトを追加してください！

2. `_template.mdx` を参考にフロントマターを設定
3. プロジェクトの詳細を Markdown で記述
4. サムネイル画像を `public/images/projects/` に追加
